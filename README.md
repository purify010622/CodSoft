# CodSoft Internship Projects

This repository contains all the projects completed during my CodSoft internship. Each project demonstrates different programming concepts and technologies.

## Projects Overview

### 1. Calculator
A simple calculator application built with Python.
- **Location**: `calculator/`
- **Features**: Basic arithmetic operations
- **Technologies**: Python, Tkinter

### 2. Contact Book
A comprehensive contact management system with authentication and advanced features.
- **Location**: `contact book/`
- **Features**: User authentication, contact CRUD operations, search functionality
- **Technologies**: Python, SQLite, Tkinter

### 3. Password Generator
A secure password generator with customizable options.
- **Location**: `password generator/`
- **Features**: Customizable length, character sets, security options
- **Technologies**: Python

### 4. Rock Paper Scissors
A web-based Rock Paper Scissors game with real-time multiplayer support.
- **Location**: `rock-paper-scissors/`
- **Features**: Real-time gameplay, multiplayer support, web interface
- **Technologies**: Flask, Socket.IO, HTML/CSS/JavaScript

### 5. To-Do List
A task management application with a modern web interface.
- **Location**: `to do list/`
- **Features**: Task creation, editing, deletion, status tracking
- **Technologies**: Python, Flask, HTML/CSS/JavaScript

## Getting Started

Each project has its own README.md file with specific setup instructions. Navigate to the respective project folder for detailed information.

## Requirements

Most projects require Python 3.7+ and their specific dependencies listed in `requirements.txt` files.

## Installation

1. Clone this repository
2. Navigate to the specific project folder
3. Install dependencies: `pip install -r requirements.txt`
4. Follow project-specific instructions in each README
